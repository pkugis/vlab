About the Courses section

Our Courses section will implement all the following functionalities:

1. List all the courses that have been registered in the system in table format. Create multiple pages if it demands....

2. All the courses can be ordered by CourseID, CourseRegisterTime, etc. (Optional)

3. Administrator can search for some course by its CourseName, CourseID, TeacherName, TeacherID, etc. Autocomplete the query if possible!

4. List all available results that match the query

5. For each course record, add certain links to some fileds. For example: Yingwei Luo, Introduction to algorithms.

6. Administrator can one course and fulfill the more detail informations about the course

7. When have access to the more detail page about a course, the administrator can the course or the detail informations about it.

8. On the detail page, administrator can also upload a selective file and generate the relationships between each student account and a certain course.

About the Teachers section

Our Teachers section will implement all the following functionalities:

1. List all the teachers that have been registered in the system in table format. Create multiple pages if it demands.

2. For each record, add the detail page link to the teachername field for example:Yingwei Luo.

3. Administrator can also and / existing teacher accounts.

4. List all the courses taught by the teacher on the detail page.

5. List all the detail information about the teacher on the detail page.
